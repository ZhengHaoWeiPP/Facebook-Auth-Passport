

## Install

Install with composer...  `composer require haowei/laravel-passport-facebook-auth`

### Versions

* Laravel 5.4 and Passport 2.0 only supported at this time

## Dependencies:
* `"laravel/passport": "^4.0"`
* `"facebook/graph-sdk": "^5.6"`
